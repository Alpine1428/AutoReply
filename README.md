# HolyWorld AutoReply v2.3

Помощник модератора HolyWorld (Fabric 1.20.1).

## Возможности
- 60+ правил умного диалога с игроком
- Авто-бан: Неадекват (30д), Отказ (30д), Признание (20д)
- Команды бана БЕЗ ника: /hm sban 30d Отказ
- Контекстный отказ: если попросили скачать AnyDesk а игрок говорит "нет"
- Если игрок говорит "у меня (название чита)" -> предлагает признание
- При /hm sban завершает проверку + endcheckout
- Авто-внос: Репорты / Проверка (взаимоисключение)
- GUI меню (ПРАВЫЙ SHIFT)

## Команды
- ПРАВЫЙ SHIFT - меню
- /ai start|stop|status|clear|test
