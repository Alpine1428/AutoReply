# HolyWorld AutoReply v2.2

Помощник модератора для сервера HolyWorld (Fabric 1.20.1).

## Возможности
- Авто-ответы на 60+ ситуаций
- Авто-бан: Неадекват (30д), Отказ (30д), Признание (20д)
- Отслеживание проверки: /hm spyfrz -> [CHECK] -> /hm sban
- GUI меню (ПРАВЫЙ SHIFT)
- Авто-внос проверок (Репорты/Проверка)
- Авто-выход (endcheckout)

## Управление
- ПРАВЫЙ SHIFT - меню
- /ai start/stop/status/clear/test

## Команды банов (без ника)
- /hm sban 30d Неадекват
- /hm sban 30d Отказ
- /hm sban 20d Признание
