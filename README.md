1. Установите [Fabric Loader](https://fabricmc.net/use/installer/)

4. Поместите оба jar в папку .minecraft/mods/
5. Запустите игру

## Возможности

### Умный автоответчик

Мод анализирует сообщения игрока и отвечает контекстно, имитируя живого модератора.

**Пример диалога:**

> Игрок: привет
>
> Бот: Привет! Проверка на читы. 7 минут. Качай AnyDesk (anydesk com), кидай код в ЛС.
>
> Игрок: а за что проверка?
>
> Бот: На тебя поступили жалобы. Качай AnyDesk - anydesk com.
>
> Игрок: не скачивается
>
> Бот: Попробуй RustDesk. Сайт: rustdesk com.
>
> Игрок: 847 293 185
>
> Бот: Код принят. Подключаюсь. Прими запрос в RustDesk.

### Автоматические баны

| Триггер | Срок | Условие |
|---------|------|---------|
| Оскорбление модератора | 30 дней | Мат направлен на человека |
| Отказ от проверки | 30 дней | Игрок написал Отказ |
| Признание в читах | 20 дней | Игрок написал Я чит |

Мат без направленности на модератора НЕ банит, а выдает предупреждение.

### Авто-переключение на RustDesk

Если AnyDesk не скачивается, мод автоматически предлагает RustDesk и дальше ведет диалог уже про него.

### Перехват команд

Мод перехватывает команды от любых источников (игрок, другие моды):
- /hm spy, /hm spyfrz - начало проверки
- /hm sban - бан
- /banip - бан (от других модов)
- /hm unfrz - разморозка

## Использование

### Быстрый старт

1. Нажмите ПРАВЫЙ SHIFT - откроется меню
2. Убедитесь что Главный переключатель = ВКЛ
3. Используйте /hm spy ник для запоминания ника
4. Используйте /hm spyfrz для начала проверки
5. Мод автоматически начнет отвечать игроку

### Порядок проверки
/hm spy ник <- запоминает ник
/hm spyfrz <- начинает ожидание
|
Игрок пишет в [CHECK] чат
|
Мод автоматически отвечает
|
Игрок скачивает AnyDesk/RustDesk
Игрок кидает код -> мод принимает
|
ИЛИ: Игрок пишет Отказ -> авто-бан 30д
ИЛИ: Игрок пишет Я чит -> авто-бан 20д
ИЛИ: Игрок оскорбляет -> авто-бан 30д
|
Проверка завершена

## Меню настроек

Открывается по нажатию ПРАВЫЙ SHIFT.

| Настройка | Описание |
|-----------|----------|
| Главный переключатель | Вкл/выкл весь мод |
| Авто-ответы | Автоматические ответы игрокам |
| Бан: Неадекват (30д) | Авто-бан за оскорбление модератора |
| Бан: Отказ (30д) | Авто-бан при написании Отказ |
| Бан: Признание (20д) | Авто-бан при написании Я чит |
| Авто-внос: Репорты | Автоматический /hm startcheckout ник report |
| Авто-внос: Проверка | Автоматический /hm startcheckout ник checkout |
| Авто-выход (endcheckout) | Автоматический /hm endcheckout после бана |

## Как работает ИИ

### Система правил

Мод использует систему приоритетных правил. Каждое правило имеет приоритет (0-500), условие и ответ.
500: Прямой отказ/признание -> Бан
400: Оскорбление модератора -> Бан
355: Контекстный отказ -> Предупреждение
305: Контекстное признание -> Предупреждение
250: Угрозы -> Ответ
200: Уход из игры -> Предупреждение
150: Код AnyDesk/RustDesk -> Принятие
130: Вопросы за что? -> Объяснение
110: Проблемы со скачиванием -> RustDesk
86: Простые ответы (да/нет) -> Контекст
15: Мат без направленности -> Предупреждение
10: Приветствие -> Начало проверки
0: Все остальное -> Напоминание

### Состояние игрока

Мод отслеживает состояние каждого игрока:

- msgCount - количество сообщений
- sentGreeting - приветствие отправлено
- askedForProgram - попросили скачать
- gaveCodes - код получен
- useRustdesk - переключились на RustDesk
- warnedRefusal - предупреждение об отказе
- warnedConfession - предупреждение о признании
- remaining() - осталось минут
- elapsed() - прошло минут

## Автоматические действия

### При /hm spyfrz

1. Состояние переходит в Ожидание CHECK
2. Если Авто-внос Репорты ВКЛ: отправляется /hm startcheckout ник report
3. Если Авто-внос Проверка ВКЛ: отправляется /hm startcheckout ник checkout

### При бане (авто или ручном)

1. Отправляется /hm sban срок причина
2. Состояние переходит в Свободен
3. Если Авто-выход ВКЛ: отправляется /hm endcheckout ban ник false

### При /banip (от другого мода)

1. Перехват через Mixin на sendChatCommand()
2. Состояние переходит в Свободен
3. Если Авто-выход ВКЛ: отправляется /hm endcheckout ban ник false

### Защита от рекурсии

Когда мод сам отправляет команду:
1. Устанавливается флаг ignoring = true
2. Миксины проверяют isIgnoring() и пропускают
3. Через 200мс флаг сбрасывается
